## Subject

```BASH
Assignment name  : aff_a
Expected files   : aff_a.c
Allowed functions: write
--------------------------------------------------------------------------------

Write a program that takes a string and displays the first character 'a' found in it, followed by a newline.

If the string does not contain the character 'a', the program simply outputs a newline.

If the number of parameters is not equal to 1, the program displays 'a' followed by a newline.

Examples:

$> ./aff_a "abc" | cat -e
a$

$> ./aff_a "dubO a POIL" | cat -e
a$

$> ./aff_a "zz sent le poney" | cat -e
$

$> ./aff_a | cat -e
a$
```
